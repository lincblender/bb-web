# bb-web

BidBlender **marketing site** — bidblender.com.

## Overview

Static marketing site with product pages, pricing, integrations, resources, and CTAs that link to the app at **bidblender.io**.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
